Guía de Referencia Rápida de HTML

Esta guía contiene una tabla organizada con los elementos, atributos y conceptos más comunes de HTML para ayudarte en tu carrera como desarrollador Full Stack.

Atributos Globales

Estos atributos se pueden usar en casi todos los elementos HTML.

Atributo Descripción Ejemplo de Uso
id Define un identificador único para un elemento. Se usa para apuntar a un elemento específico con CSS o JavaScript. No debe repetirse en la página. <div id="main-container">...</div>
class Asigna una o más clases a un elemento. Se usa para aplicar estilos CSS o seleccionar múltiples elementos con JavaScript. Un elemento puede tener varias clases, separadas por espacios. <p class="text-bold featured">...</p>
style Aplica estilos CSS en línea directamente al elemento. No es una práctica recomendada para estilos complejos, es mejor usar archivos CSS externos. <h1 style="color: blue; font-size: 24px;">Título</h1>
title Proporciona información adicional sobre un elemento. Generalmente se muestra como un "tooltip" (cajita de texto) cuando el cursor pasa sobre el elemento. <abbr title="HyperText Markup Language">HTML</abbr>
lang Especifica el idioma del contenido del elemento. Ayuda a los motores de búsqueda y navegadores. <p lang="fr">Bonjour le monde!</p>
data-* Permite almacenar datos personalizados privados para la página o aplicación. Son completamente personalizables (ej. data-user-id). <div data-product-id="12345">Producto</div>

Elementos de Estructura y Contenido

Estos elementos definen la estructura semántica de la página y organizan el contenido principal.

Elemento Descripción Atributos y Ejemplo de Uso
<html> Elemento raíz que envuelve todo el contenido de la página. lang: Especifica el idioma.
<html lang="es">...</html>
<head> Contiene metadatos del documento, como el título, enlaces a CSS y scripts. No es visible en la página. <head>...</head>
<body> Contiene todo el contenido visible de la página web. <body>...</body>
<header> Define la cabecera de una página o sección. Suele contener el logo, el título principal y la navegación. <header><h1>Mi Blog</h1></header>
<nav> Define un bloque de enlaces de navegación. <nav><a href="/inicio">Inicio</a></nav>
<main> Representa el contenido principal y único del documento. Solo debe haber un <main> por página. <main><h2>Artículo Principal</h2></main>
<section> Define una sección genérica en un documento, generalmente con un encabezado. <section><h2>Sobre Nosotros</h2>...</section>
<article> Representa contenido independiente y autocontenido, como un post de blog, un artículo de noticias o un comentario. <article><h3>Título del Post</h3>...</article>
<aside> Define contenido relacionado pero separado del contenido principal (como una barra lateral o un bloque de anuncios). <aside><h3>Posts Relacionados</h3>...</aside>
<footer> Define el pie de página de un documento o sección. Suele contener información de copyright, contacto y enlaces. <footer><p>© 2024 Mi Empresa</p></footer>
<h1> a <h6> Definen encabezados, donde <h1> es el más importante y <h6> el menos importante. <h1>Título Principal</h1>
<p> Define un párrafo de texto. <p>Este es un párrafo.</p>
<a> Crea un hipervínculo (enlace) a otras páginas web, archivos o ubicaciones dentro de la misma página. href: URL de destino.
target="_blank": Abre en nueva pestaña.
<a href="https://google.com">Ir a Google</a>
<img> Incrusta una imagen en la página. Es un elemento vacío (no tiene etiqueta de cierre). src: Ruta de la imagen.
alt: Texto alternativo (importante para accesibilidad).
<img src="logo.png" alt="Logo de la empresa">
<ul>, <ol>, <li> <ul>: Lista desordenada (viñetas).
<ol>: Lista ordenada (números).
<li>: Elemento de una lista.		 <ul><li>Elemento 1</li></ul>
<div> Contenedor genérico en bloque. Se usa para agrupar otros elementos y aplicarles estilos con CSS o manipularlos con JavaScript. No tiene valor semántico. <div class="container">...</div>
<span> Contenedor genérico en línea. Se usa para agrupar texto o elementos en línea para aplicar estilos o manipularlos. No tiene valor semántico. <p>Texto con <span style="color:red;">énfasis</span>.</p>

Elementos de Formularios

Estos elementos se utilizan para recopilar datos del usuario.

Elemento Descripción Atributos y Ejemplo de Uso
<form> Contenedor para controles de formulario. action: URL a la que se envían los datos.
method: Método HTTP (get o post).
<form action="/submit" method="post">...</form>
<input> Elemento de entrada de datos más versátil. Su comportamiento cambia según el atributo type. type: (text, password, email, number, checkbox, radio, submit, etc.).
name: Nombre del campo.
value: Valor del campo.
placeholder: Texto de ejemplo.
required: Campo obligatorio.
<input type="text" name="username" placeholder="Nombre de usuario">
<label> Define una etiqueta para un elemento <input>. Mejora la accesibilidad. for: Asocia la etiqueta con un input a través de su id.
<label for="user">Usuario:</label><input id="user" type="text">
<textarea> Define un campo de entrada de texto de múltiples líneas. rows: Número de filas visibles.
cols: Número de columnas visibles.
<textarea name="comment" rows="4" cols="50"></textarea>
<select> y <option> <select>: Crea un menú desplegable.
<option>: Define una opción dentro del menú.		 <select name="role"><option value="admin">Administrador</option></select>
<button> Define un botón clicable. type: (submit, reset, button).
<button type="submit">Enviar</button>
<fieldset> y <legend> <fieldset>: Agrupa elementos relacionados en un formulario.
<legend>: Define un título para el <fieldset>.		 <fieldset><legend>Información Personal</legend>...</fieldset>
