Historia y proposito

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Fue desarrollado por Tim Berners-Lee en 1991 y ha evolucionado a lo largo de los añospara incluir nuevas características y capacidades.

Ejemplo de Estructura Basica de HTML

                    
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Primera Pagina Web</title>
